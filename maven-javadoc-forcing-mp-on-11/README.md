```
mvn clean package
```

* This project has an automatic module name, it is therefore named but non modular.
  Modularized apps can refer to the module on the module path, but the project cannot take
  advantage from the module path.

```java
  boolean supportModulePath = javadocRuntimeVersion.isAtLeast( "9" )
            && ( source == null || JavaVersion.parse( source ).isAtLeast( "9" ) )
            && ( release == null || JavaVersion.parse( release ).isAtLeast( "9" ) );
```
